# SalesTeamHelper_bot
@imagerfrobot - Telegram bot for team "in fields", allowing quickly notify responsible persons about any issues in sales points

Steps: sales team person comes into some sales point - shop etc - and see some issues, like goods misplacement. Than he start bot, mark address and distributor (shop) info, select problem type, add photo and comments - and bot sends e-mail with all that data to shop curator.

@imagerfrobot is active and can be tested at any time.
